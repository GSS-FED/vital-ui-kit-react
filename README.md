## Usage

### Installation
```bash
# yarn
yarn add @gssfed/vital-ui-kit-react

# npm
npm i --save @gssfed/vital-ui-kit-react
```
Import packages you want to use

`import { Button } from '@gssfed/vital-ui-kit-react`



## Development

`yarn install`

then run storybook

`yarn run storybook`

## Build

`yarn run build-storybook`
