## Usage

### Installation
```
yarn add @gssfed/vital-ui-kit-react
// or
npm i --save @gssfed/vital-ui-kit-react
```

Import packages you need

```js
import { Button } from '@gssfed/vital-ui-kit-react'

/* ... */
render() {
  return (
    <Button>Vital 💜 React!</Button>
  )
}
````



## Development

`yarn install`

then run storybook

`yarn run storybook`

## Build

`yarn run build-storybook`
